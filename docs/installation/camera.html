2. ZED camera¶

We use ZED camera as our visual input.

2.1. Install cv-bridge¶

Install cv-brdige package through apt-get.

sudo apt update
sudo apt install -y ros-melodic-cv-bridge
sudo apt install -y ros-melodic-vision-opencv
